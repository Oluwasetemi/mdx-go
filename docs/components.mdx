# Components

## Head

Renders head content

```mdx
import { Head } from 'mdx-go'

<Head>
  <title>Hello Title</title>
</Head>

# Page with title
```

## Link

Link component for handling client-side routing.
This component is used as the default `<a>` tag in MDX to support routing within react-router.

```jsx
import { Link } from 'mdx-go'
```
